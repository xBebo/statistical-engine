# 📊 Statistical Engine
A high-performance statistical calculator built with **React** and **FastAPI**. Designed to instantly calculate and visualize Confidence Intervals and Hypothesis Testing.

## 🚀 Features
*   **Dynamic Distributions:** Renders Z, T, and Chi-Square curves in real-time.
*   **Textbook Accuracy:** Detailed "Lecture Method" breakdowns for step-by-step verification.
*   **Smart Visualization:** Conditional shading for Confidence Intervals vs. Hypothesis Rejection regions.
*   **Zero-Centered Curves:** Standardized Z-score plotting for consistent academic visualization.

## 🛠️ Tech Stack
*   **Frontend:** React, Tailwind CSS, Recharts
*   **Backend:** Python, FastAPI, Scipy, Numpy

## 🔧 Local Installation
1. Clone the repository.
2. **Backend:** `cd backend`, activate your virtual environment, run `pip install -r requirements.txt`, and start the server with `uvicorn main:app --reload`.
3. **Frontend:** `cd frontend`, run `npm install`, and start the UI with `npm run dev`.
